## Consolidated feature inventory (product)
1. Menu-bar-only voice companion with a **named mascot persona** (Clicky).
2. **Push-to-talk voice** → talk to AI, spawn agents. 0 setup, consumer-grade.
3. **Voice computer control** — "no hands" operation (GPT-Realtime era → now Claude Fable 5 default).
4. **Screen awareness** — screenshots as context; the model sees your screen.
5. **Screen drawing / pointing** — blue cursor evolved into drawing polygons + pixel-perfect pointing = a live on-screen tutor (Claude Opus for the drawing/pointing).
6. **Spatial context (draw-to-direct)** — user draws on their own screen to tell the AI where to look.
7. **Screen-aware dictation** — fast STT (~450ms) that writes into the focused app using screen context (Claude Code prompt, Gmail reply in your voice).
8. **Builds Mac apps**, **research reports** (IG micro-influencers), **dev setup doctor** (install Claude Code), **teaches** (Figma motion, FL Studio, Pythagorean).
9. **Native + external integrations** — Apple Notes/Calendar/Reminders natively; Composio for Gmail/Notion/Linear/etc (per app bundle).
10. **Drag files in** to chat. Ambient/proactive nudging vision.

## Content strategy notes (for skill suite)
- **Dual-channel**: founder flagship (authority, "we shipped") + mascot diary (personality, relatability). The mascot tweets carry the brand's charm and keep cadence between big drops.
- **Flagship hook formula**: bold claim → acknowledge the crowded space → "so we went further" → the specific unique mechanism → "Demo:".
- **Cadence**: ~weekly big drop + frequent small mascot posts. ~11 weeks Apr→Jul.
- **Format split**: 4K landscape for founder demos; 1080x1920 vertical for mascot/social.
- **Reveals build**: pointer → drawing → spatial context → dictation. Each ships one crisp new primitive.
- Top performers: voice control (14.2k), dictation (9.9k), hero (6.1k), tutor (5.2k), spatial (2.8k), ambient (2.2k).
